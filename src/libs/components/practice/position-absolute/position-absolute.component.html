Абсолютне позиціонування

Контейнер - шапка
Контейнер - колонтитул 
Елемент у котрого абсолютне позиціонування видаляється зі звичайного документообігу, і для нього не створюється місця в макеті сторінки.
Він розташовується відносно батьківсього елемента (у котрого position !== 'static').
Кінцеве положення такого елемента визначається значеннями top, right, bottom, left.
Необов'язково задавати значення для всіх чотирьох сторін, зазвичай достатньо для top та left або right.
